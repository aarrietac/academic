Virtual Test Axle Kinematics

Description

The virtual test axle kinematics is a tool programmed in python that helps the understanding of how works the suspension systems. In this tool, the Double Wishbone, McPherson and Multi-link axle kinematics are available.

About the modeling

The kinematic model of each axle is based on the book, Road Vehicle Dynamics: Fundamentals and Modeling
